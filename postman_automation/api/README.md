# Postman коллекция

Готовая коллекция с автоматическими тестами и request chaining.

## Как использовать

1. Postman → **Import** → выберите `postman_collection.json`
2. Postman → **Import** → выберите `postman_environment.json`
3. Откройте коллекцию "Task Manager API - Comlete Test Suite"
4. Нажмите **Run collection**
5. Нажмите **Run**
