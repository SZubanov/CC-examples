# Postman коллекция

Готовая коллекция с автоматическими тестами и request chaining.

## Как использовать

1. Postman → **Import** → выберите `collection-example.json`
2. Откройте коллекцию "Task Manager API - Example Collection"
3. Нажмите **Run collection**
4. Нажмите **Run**

Готово! Все переменные подставляются автоматически.

## Что проверяют тесты

- Статус коды ответов (200, 201, 401, 404)
- Структура JSON ответов
- Наличие обязательных полей
- Автоматическое сохранение `auth_token`, `user_id`, `task_id`
